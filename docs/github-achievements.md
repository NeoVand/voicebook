# Legitimate GitHub achievement track

GitHub describes achievements as a public-preview feature whose criteria may change. Voicebook only pursues activity that improves the project or another real user’s work.

- Pull Shark: the issue-linked implementation sequence creates reviewable, coherent PRs rather than empty churn.
- Pair Extraordinaire: only add a coauthor when a human materially designs, codes, tests, documents, or reviews the merged change and consents to attribution.
- Galaxy Brain: answer reproducible Svelte/SvelteKit Discussions with tested minimal examples, then continue supporting the asker. Acceptance belongs to the asker and cannot be guaranteed.
- Public Sponsor: if desired, publicly sponsor [Transformers.js creator Xenova](https://github.com/sponsors/xenova). The account owner chooses the tier and completes checkout; Voicebook never automates payment.

Do not manufacture comments, close throwaway issues, create fake coauthors, or ask someone to accept an answer solely for a badge. GitHub’s own documentation notes that achievements are public-preview profile events, while sponsorships can be public or private and involve a real charge.
